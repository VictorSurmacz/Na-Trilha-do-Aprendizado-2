Table

Essa tag ela cria uma tabela, o que usaremos é (table, tr, th e td). O table é o que vai definir que faremos uma tabela, o tr delimita linha da tabela, o th define o titulo da tabela e o td o conteúdo da tabela.Também exitem as colspan, define quantos espaços a informação tomara na tabela. rowspan ocupa celulas para baixo da tabela.

"del"(ou Elemento HTML de Texto Excluído) representa uma parte do texto que foi excluída de um documento. Este elemento é (não necessariamente) renderizado pelos navegadores com uma linha entre o texto.

INS

representa um intervalo de texto que foi adicionado a um documento.

SUB

define o texto subscrito. O texto subscrito aparece meio caractere abaixo da linha normal e, às vezes, é renderizado em uma fonte menor.

SUP

texto sobrescrito. O texto sobrescrito aparece meio caractere acima da linha normal e, às vezes, é renderizado em uma fonte menor.

BLOCKQUOTE

indica que o texto incluído é uma longa citação.

Q

indica que o texto dentro da tag é uma pequena citação.

ADDR

representa uma abreviação e opcionalmente fornece uma descrição completa para ela. Se presente, o atributo title deve conter a descrição completa e apenas ela.

ADDRESS

fornece informações de contato para seu ancestral article ou body mais próximo; no segundo caso, ele se aplica ao documento inteiro.

CITE

representa uma referência a um trabalho artístico. Deve incluir o título do trabalho ou uma URL de referência

BDO

é usado para substituir a direcionalidade atual do texto.

CODE

tag é usada para definir um pedaço de código de computador.

PRE

é uma tag utilizada para representar texto pré-formatado.

KBD

tag é usada para definir a entrada do teclado.
